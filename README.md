Чтобы файл запускать на рабочем столе, вам нужно создать .bat файл. Делается это очень просто:
1) Создаете текстовый документ на рабочем столе
2) Пишите туда python.exe C:\путь\до\файла\main.py
3) Сохраняете и меняете расширение документа с .txt на bat

Теперь у вас будет понижаться приоритет задач для дискорда и повышаться для Geometry dash
P.s. Возможно вам потребуется установить библиотеку используемую в программе.
